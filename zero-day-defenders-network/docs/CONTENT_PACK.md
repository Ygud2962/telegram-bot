## 4. Математика баланса

### Энергия

- 12 энергии в день.
- 1 энергия = 1 попытка с наградой.
- Средний игрок закрывает 8-10 попыток.
- Hardcore игрок закрывает 12 и идет в сюжет/отряд.
- SOC ELITE дает 1 час безлимита, но результаты сверх лимита не повышают fair score.

### Кредиты

Базовая формула:

```
credits = baseThreatReward * difficultyMultiplier * accuracyMultiplier * comboMultiplier
```

Где:

- `baseThreatReward`: 60-180;
- `difficultyMultiplier`: 1.0-2.5;
- `accuracyMultiplier`: 0.5-1.3;
- `comboMultiplier`: 1.0-1.5.

Средний доход без доната:

- день 1-30: 900-1400 кредитов/день;
- день 31-90: 1500-2600;
- день 91-180: 3000-5500.

### Drop карточек

После обычной угрозы:

- Common: 60%;
- Rare: 25%;
- Epic: 12%;
- Legendary: 3%.

После daily combo:

- Rare: 70%;
- Epic: 27%;
- Legendary: 3%.

После boss:

- Rare: 50%;
- Epic: 40%;
- Legendary: 10%.

### Pity

```
rarePity += 1
epicPity += 1
legendaryPity += 1

if legendaryPity >= 100: force Legendary
else if epicPity >= 50: force Epic
else if rarePity >= 10: force Rare
```

### Прокачка карточек

- уровень 1: карточка открыта;
- уровень 2: +1 дубль;
- уровень 3: +3 дубля;
- MAX: +7 дублей;
- Holo: MAX + сезонный материал.
