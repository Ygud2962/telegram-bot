## 🎮 Игра "ШИФРОВАЛЬЩИК · 1941-1945"

**Telegram Mini App** — образовательная игра о событиях Великой Отечественной войны на территории Беларуси

### Особенности игры:
- 🔐 **6 глав** — от Хойников до Берлина
- 🎯 **36 заданий** — по 6 шифров в каждой главе
- 🧩 **6 типов шифров**: Цезарь, Морзе, Атбаш, Числовой, Анаграмма, Математика
- ❤️ **5 жизней** на каждую главу
- ⚡ **Очки за скорость** — чем быстрее, тем больше
- 🏆 **Таблица лидеров** всей школы
- 🎖 **50+ достижений** для настоящих разведчиков
