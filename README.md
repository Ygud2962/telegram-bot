## ⚡ Возможности

| | |
|---|---|
| ### 📚 Расписание уроков  
Классы 5а–11, пн–пт. На день, неделю или прямо **сейчас** — с заменами в реальном времени  

### 👨‍ Расписание учителей  
Поиск, пагинация, авторегистрация через `/teacher` — уведомления при заменах автоматически  

### 🔄 Замены уроков  
Вчера / сегодня / завтра. Добавление вручную или **через фото** с ИИ-распознаванием  

### 📣 Школьные новости  
3 последних на главном экране, архив с пагинацией, рассылка всем в один клик  

### 🌟 Избранное «МОЁ»  
Добавь свой класс и учителей — всё важное доступно с первого экрана  

### 🤖 ИИ-помощник  
Groq `llama-3.3-70b`, память диалога, два режима: педагог и ученик  

### ⏰ Сейчас / Звонки  
Текущий урок, время до звонка, следующий предмет — живое обновление  

### 🔍 Поиск учителя  
По фамилии или части имени, мгновенные результаты |

## 🎮 Игра "ШИФРОВАЛЬЩИК · 1941-1945"

**Telegram Mini App** — образовательная игра о событиях Великой Отечественной войны на территории Беларуси

### Особенности игры:
- 🔐 **6 глав** — от Хойников до Берлина
- 🎯 **36 заданий** — по 6 шифров в каждой главе
- 🧩 **6 типов шифров**: Цезарь, Морзе, Атбаш, Числовой, Анаграмма, Математика
- ❤️ **5 жизней** на каждую главу
- ⚡ **Очки за скорость** — чем быстрее, тем больше
- 🏆 **Таблица лидеров** всей школы
- 🎖 **50+ достижений** для настоящих разведчиков

### 🚀 Играть сейчас:
🔗 **[Открыть игру](https://ygud2962.github.io/telegram-bot/)**  
или через бота: кнопка **"🎮 Шифровальщик"**
