## 🎮 Режимы доступа игры
Глобальный режим включается в админ-панели `Управление игрой`:

| Режим | Поведение |
|---|---|
| `closed` | ❌ Доступ к игре закрыт всем игрокам |
| `beta` | 🧪 Доступ только разрешённым (бета-список / tester / admin / bot-admin) |
| `open` | ✅ Доступ открыт всем |

> Смена режима: `Бета-режим` / `Игра открыта` / `Игра закрыта`.

## 🛠 Установка и запуск

### Локальная разработка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ygud2962/telegram-bot.git
cd telegram-bot

# 2. Создайте виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Создайте .env файл
cp .env.example .env
# Отредактируйте переменные окружения

# 5. Запустите бота
python bot.py
```

### Деплой на Railway

```bash
# 1. Подключите репозиторий к Railway
# 2. Добавьте переменные окружения в настройках проекта
# 3. Railway автоматически запустит деплой через Procfile

# Переменные для Railway:
BOT_TOKEN=your_telegram_bot_token
DATABASE_URL=postgresql://user:pass@host:port/db
GROQ_API_KEY=your_groq_api_key  # опционально
GAME_URL=https://ygud2962.github.io/telegram-bot/
BOT_PUBLIC_URL=https://your-app.up.railway.app
```

### Деплой игры на GitHub Pages

```bash
# 1. В настройках репозитория → Pages
# 2. Выберите ветку: main, папка: /game
# 3. GitHub автоматически опубликует игру
# 4. URL будет: https://username.github.io/repo-name/
```
