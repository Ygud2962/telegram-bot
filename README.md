### 📚 Расписание уроков
Классы 5а–11, пн–пт. На день, неделю или прямо **сейчас** — с заменами в реальном времени

### 👨‍🏫 Расписание учителей
Поиск, пагинация, авторегистрация через `/teacher` — уведомления при заменах автоматически

## 👑 Админ-панель

| Функция | Описание | Статус |
|---------|----------|:------:|
| 📸 Замены из фото | Фото листа → ИИ читает → предпросмотр → сохранение | 🟢 `NEW` |
| 📣 Публикация новостей | Предпросмотр + мгновенная рассылка всем | 🔥 `HOT` |
| ➕ Добавление замен вручную | 4 шага: дата → класс → урок → учитель | 🟢 Готово |
| 🤖 Groq Vision | ИИ читает фото и распознаёт замены | 🟣 `AI` |
| 📊 Аналитика | Активность, популярные классы, пиковые часы | 🟢 Готово |
| 🔧 Техрежим | 1ч / 3ч / 5ч / До завтра / Бессрочно | 🟢 Готово |
| 👥 Статистика | Пользователи, логи, активность | 🟢 Готово |
