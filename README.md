## 📋 О проекте

> 🤖 *Telegram-бот для школы, который помогает ученикам, учителям и родителям всегда быть в курсе расписания уроков, замен и важных новостей.*

```
⚡ Быстро  •  🔒 Надёжно  •  📱 Удобно  •  🎯 Точно
```

## 🚀 Что умеет бот

```
📚 Расписание уроков
   └─ Все классы 5а–11 • Пн-Пт • Мгновенный доступ

👨‍🏫 Расписание учителей  
   └─ Фильтрация • Поиск • Уведомления о заменах

🔄 Замены уроков
   └─ Актуально • С уведомлениями • В 4 шага добавления

📣 Школьные новости
   └─ Мгновенная рассылка • Важные объявления • История

🌟 Кнопка «МОё»
   └─ Избранное • Быстрый доступ • Личное пространство

⏰ Раздел «Сейчас»
   └─ Текущий урок • Время до звонка • Живое обновление

🔔 Расписание звонков
   └─ Уроки • Питание • Перемены • Всё по минутам

🔍 Поиск учителя
   └─ По фамилии • Мгновенно • Без лишних кликов
```

## ⚡ Как начать пользоваться

```
┌─────────────────────────────────┐
│  1️⃣  Найдите бота в Telegram   │
│  2️⃣  Нажмите /start            │
│  3️⃣  Выберите раздел в меню    │

```
