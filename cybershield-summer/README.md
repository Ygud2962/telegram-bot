# ZERO_DAY: Школьный Протокол — Прототип Mini App

## Структура файлов

```
zero_day/
├── index.html   — HTML-оболочка реалистичного смартфона
├── styles.css   — Все стили: корпус телефона, экраны, компоненты
├── data.js      — Весь игровой контент (эпизоды, персонажи, улики, магазин)
├── app.js       — Движок приложения: навигация, все экраны, игровая логика
└── README.md    — Этот файл
```

## Запуск

Просто открой `index.html` в браузере. Никаких зависимостей, никакого сервера — всё работает как статичный сайт.

```bash
# Вариант 1: открыть файл напрямую
open index.html

# Вариант 2: простой HTTP-сервер (для разработки)
python3 -m http.server 8080
# затем открыть http://localhost:8080
```

## Интеграция в Telegram Mini App

В `index.html` добавь перед `</head>`:
```html
<script src="https://telegram.org/js/telegram-web-app.js"></script>
```

В `app.js` добавь инициализацию:
```js
if (window.Telegram?.WebApp) {
  Telegram.WebApp.ready();
  Telegram.WebApp.expand();
  // Получить имя пользователя:
  const user = Telegram.WebApp.initDataUnsafe?.user;
  // Оплата Stars:
  Telegram.WebApp.openInvoice(url, callback);
}
```

## Игровая механика (текущий прототип)

- **Stars (⭐)** — внутренняя валюта, начальный баланс 47
- **Мессенджер** — 3 варианта выбора → реакция NPC, начисление Stars
- **Галерея** — анализ EXIF-метаданных, флаги угроз, +3⭐ за опасные файлы
- **Браузер** — 4 кликабельных признака фишинга, +5⭐ за все 4
- **Терминал** — шифр Цезаря (ответ: сдвиг 4), +8⭐ за решение
- **Магазин** — покупка реквизита, баланс уменьшается, защита от отрицательного баланса
