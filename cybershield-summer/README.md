# Киберщит — Летний сезон 2026 (отдельный проект)

Отдельная летняя игра по кибербезопасности.  
Сделана изолированно, чтобы не затрагивать текущий `Шифровальщик` и рабочую логику бота.

## Главное

- Период миссий: **01.06.2026 — 31.08.2026**.
- Формат: **92 миссии (по 1 в день)**.
- Дизайн: современный `glass/cyber` интерфейс (мобильный-first).
- Локальное сохранение прогресса (`localStorage`, ключ `cs_progress_v2`).

## Режимы

1. `🔍 Поиск улик на экране`
2. `💬 Симулятор переписки` (свободный ответ, не тест с вариантами)
3. `📥 Сортировка входящих` (безопасно / подозрительно / опасно)
4. `⚡ Инцидент 30с` (правильная последовательность действий под таймер)

## Файлы

- `index.html` — экран игры и блоки режимов.
- `styles.css` — новый современный визуал.
- `app.js` — логика миссий, режимов и прогресса.

## Изоляция от текущего проекта

1. Отдельная папка и отдельные файлы.
2. Нет импорта `game.js`, `bot.py`, `database.py`.
3. Нет вызовов `/game_sync` и прочих endpoint-ов Шифровальщика.
4. Прогресс хранится в отдельном ключе `cs_progress_v2`.

## Летний календарь

Календарь генерируется программно:

- Старт: `2026-06-01`
- Конец: `2026-08-31`
- Ротация по дням недели:
  - Пн — `clue`
  - Вт — `chat`
  - Ср — `sort`
  - Чт — `incident`
  - Пт — `clue`
  - Сб — `chat`
  - Вс — `sort`

Или запустить локальный сервер:

```powershell
cd C:\Users\uragu\telegram-bot\cybershield-summer
python -m http.server 8099
```

URL:

- [http://localhost:8099](http://localhost:8099)

## Как позже подключить в бот

1. Оставить проект отдельным (папка/домен).
2. Добавить кнопку в боте `🛡 Киберщит`.
3. Открывать отдельный URL (`/cybershield/`).
4. Для серверной синхронизации использовать отдельные endpoint-ы:
   - `/cyber_state`
   - `/cyber_sync`
   - `/cyber_leaderboard`

Так существующая игра и бот останутся стабильными.
